تعلم البرمجة بـ JavaScript — عملي وتفاعلي

دروس قصيرة بالعربي، بعد كل درس سؤال، وبعدها محرر تكتب فيه الكود ويتحقق لك إذا صحيح أو لا.

المقدمة — ما هي البرمجة؟

البرمجة هي إعطاء أوامر للكمبيوتر ليؤدي مهام. نكتب أوامر (كود) بلغة يفهمها المتصفح مثل JavaScript.

سؤال: صح أم خطأ: JavaScript تعمل فقط على الخادم (Server).

المتغيرات (Variables)

المتغير يخزن قيمة. مثال:

let name = "Mohammed";
console.log(name);

سؤال: أي سطر يُنشيء متغيراً؟

الشروط (if)

نستخدم الشروط لتنفيذ كود إذا تحقق شرط معين.

let x = 10;
if (x > 5) { console.log('big'); }

سؤال: لو x = 3، هل سيطبع 'big'؟

الحلقات (Loops)

تستخدم الحلقات لتكرار تنفيذ كتلة كود.

for (let i = 0; i < 3; i++) { console.log(i); }

سؤال: كم مرة سينفذ الكود في المثال أعلاه؟

المحرر التفاعلي — اكتب الكود وتأكد منه

في هذا المحرر نطلب منك كتابة كود يطبع نتيجة معينة. بعد كتابة الكود اضغط Run. الزر الأخضر هو زر التشغيل.
